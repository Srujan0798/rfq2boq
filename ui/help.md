# RFQ2BOQ Help

## What is RFQ2BOQ?

RFQ2BOQ extracts Bill of Quantities (BOQ) items from construction tender documents (PDFs). It automatically identifies materials, quantities, units, and specifications from uploaded RFQ files.

## How to Use

1. **Upload PDF** — Drag and drop your tender PDF or use the "Try Sample" button
2. **Wait** — Extraction takes 30-60 seconds depending on document size
3. **Review** — Check the extracted items in the table
4. **Edit** — Click any cell to make corrections
5. **Download** — Export as Excel, JSON, or CSV

## Quality Guide

| Label | Meaning | Action |
|-------|---------|--------|
| Good 🟢 | High confidence | Items look correct |
| Check 🟡 | Medium confidence | Review before use |
| Verify 🔴 | Low confidence | Manual verification needed |

## Support

For issues or questions, contact your system administrator with the document name and description of the problem.
