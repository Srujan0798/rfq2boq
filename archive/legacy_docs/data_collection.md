# Data Collection — Real RFQ PDFs

Documentation for collecting real construction RFQ PDFs for evaluation and training.

## Legal Posture

Indian government tender documents published on e-Procurement portals are public documents under the Right to Information (RTI) Act, 2005. The documents themselves are in the public domain. However:

- **Cite sources:** Always reference the portal (etenders.gov.in, cppp.nic.in, etc.) and date
- **Redact PII:** Remove bidder names, contractor details, and contact information before any sharing
- **robots.txt:** Always respect robots.txt directives; rate-limit to ≥2s between requests
- **No commercial redistribution:** Store locally; do not republish PDFs without permission

### Manual Fallback

If all portals are inaccessible:
1. Go to the portal directly in a browser
2. Search for "building construction" or "RFQ" tenders
3. Download PDFs to `data/real_rfqs/raw/`
4. Create manifest entry manually

## Scraper Usage

```bash
# Primary scrape (50 PDFs target)
python scripts/collect_real_rfqs.py --limit 50 --output data/real_rfqs/raw

# Alternative: python scripts/scrape_etenders.py --limit 50 --output data/real_rfqs/raw

# Fallback: manual download (portals often block scraping)
# Download from eprocure.gov.in, cpwd.gov.in, tntenders.gov.in
# Place PDFs in data/real_rfqs/raw/ then run:
python scripts/collect_real_rfqs.py --output data/real_rfqs/raw

# Validate manifest
python scripts/validate_manifest.py --manifest data/real_rfqs/raw/manifest.json
```

## Manifest Format

The `manifest.csv` in `data/real_rfqs/raw/` contains one row per PDF:

```csv
filename,source,file_size_kb,sha256,pages,chars,is_real,is_scanned
ireps_2724bb1eff78.pdf,ireps,1231.7,2724bb1eff7867032053313af2562c...,12,30228,True,False
rfq_building_RFQ1138_005.pdf,synthetic_building,156.3,abc123...,8,8920,False,False
```

Generated by: `python3 scripts/build_manifest.py`

## Manual Download Instructions

1. Open browser to target portal
2. Search with keywords: "building construction", "residential", "RFQ"
3. Filter to PDF documents only
4. Download to `data/real_rfqs/raw/`
5. Run: `python scripts/generate_manifest.py --directory data/real_rfqs/raw`

## Processing After Collection

```bash
# Process all collected PDFs
python scripts/process_real_rfqs.py --input data/real_rfqs/raw --output data/real_rfqs/extracted

# Gold annotate subset (10 PDFs)
python scripts/annotate_data.py --input data/real_rfqs/raw --output data/real_rfqs/test_gold --count 10

# Re-evaluate model on real data
python scripts/evaluate_real.py --gold data/real_rfqs/test_gold --output results/real_world_metrics_v2.json
```

## Reproducibility

- Document all portal URLs and access dates
- Store SHA256 of every PDF for integrity verification
- Keep scrape logs for audit trail
- Use Python 3.11–3.13 (avoid Python 3.14 typer/click bug)
