# TASK: HuggingFace Dataset Release — Agent-1

**Wave:** 5 | **Tier:** D | **Priority:** P3

## 1. GOAL
Release the RFQ2BOQ NER dataset on HuggingFace Datasets as a public benchmark for construction NLP research.

## 2. CONTEXT
Read first:
- `data/annotations/` — current BIOES annotations
- `data/real_rfqs/` — real-world annotations
- [docs/conventions.md](../../../../docs/conventions.md)

Current state: Annotations are project-local. No public release.

## 3. DELIVERABLES
- [ ] `data/RFQ-BOQ-NER/` — packaged dataset directory
- [ ] `data/RFQ-BOQ-NER/README.md` — data card
- [ ] `data/RFQ-BOQ-NER/dataset_infos.json` — HF metadata
- [ ] `data/RFQ-BOQ-NER/rfq_boq_ner.py` — HF loading script
- [ ] `data/RFQ-BOQ-NER/train.jsonl`, `val.jsonl`, `test.jsonl` — CoNLL/JSON format
- [ ] `data/RFQ-BOQ-NER/CITATION.bib` — BibTeX
- [ ] `scripts/upload_dataset.py` — push to HF Hub
- [ ] `scripts/benchmark.py` — standardized eval

## 4. STEPS
1. Convert annotations to CoNLL format + JSONL (compatible with most NER tools)
2. Train/val/test splits documented and reproducible
3. Data card includes: description, collection process, language(s), licensing (CC-BY-SA 4.0), citation
4. Loading script for HF Datasets library
5. Benchmark script: load → run model → compute span F1 → output leaderboard-friendly JSON
6. Upload via `huggingface-cli login` + `dataset.push_to_hub("[username]/rfq-boq-ner")`
