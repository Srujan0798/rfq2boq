## Step 2: Prepare Provisional Patent Application

A provisional patent provides 12 months of protection while you assess commercial value.

### Required Documents:

1. **Provisional Patent Specification (30-50 pages)**
   - Title: "System and Method for Automated Bill of Quantities Extraction from Construction Tender Documents using Hybrid Machine Learning and Domain Ontology"
   - Field of Invention
   - Background (prior art deficiencies)
   - Summary of Invention
   - Detailed Description (with figures)
   - Claims (at least independent claims)
   - Abstract (150 words)

2. **Inventor Declaration Form**
   - All named inventors must sign
   - Include contribution percentage for each

3. **Date of Invention**
   - Document the earliest conception date
   - Laboratory notebooks / git commit dates as evidence

4. **Drawings/Figures (5-10)**
   - Pipeline architecture diagram
   - Ontology structure diagram
   - Entity resolution flow chart
   - Sample RFQ input
   - Sample BOQ output

## Step 3: File Provisional Application

### Option A: Through IIT Hyderabad TTO (Recommended)
1. Contact TTO with invention disclosure
2. TTO conducts prior art search
3. If approved, TTO files on your behalf
4. Cost: Typically ₹10,000-50,000 (university subsidized)

### Option B: Self-File Online
1. Register at: https://ipindia.gov.in
2. Fill Form 1 (Provisional Application)
3. Pay fees: ₹1,600 (individual, natural)
4. Submit via online portal or at Patent Office
