## Invention Title

**"System and Method for Automated Bill of Quantities Extraction from Construction Tender Documents using Hybrid Machine Learning and Domain Ontology"**

## Brief Description

I have developed an NLP system that automatically extracts structured Bill of Quantities (BOQ) from unstructured construction tender PDF documents (RFQs). The system uses a hybrid approach combining:

1. **BERT-BiLSTM-CRF NER** with BIOES tagging for entity recognition
2. **Domain-specific ontology** (249+ construction materials) for validation
3. **LLM-based ambiguity resolution** for low-confidence predictions
4. **Active learning loop** for continuous model improvement

## Key Innovations

1. **Hybrid ML + Rule Conflict Resolution Algorithm** - When BERT confidence < 0.5, the system queries an LLM for ambiguity resolution, improving extraction accuracy.

2. **BIOES-Tagged Synthetic RFQ Generator** - Generates synthetic training data using construction-specific templates with proper BIOES tagging schemes.

3. **Construction-Specific Entity Ontology** - 249+ construction materials organized hierarchically with attributes including typical units, grade ranges, and standard references.

4. **Real-Time Uncertainty-Driven Active Learning Loop** - Identifies uncertain predictions for human review and incorporates corrections back into training data.

## Performance Metrics

- **Synthetic F1:** 99.6%
- **Real-world F1:** 67.05% (on 60 real government tender PDFs)
- **Processing Speed:** ~4 seconds per page
